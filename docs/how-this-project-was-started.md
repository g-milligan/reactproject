# How this project was started

[Documentation](https://react-redux.js.org/introduction/getting-started)

Initialize project:

`project root`:
``` shell
npx degit reduxjs/redux-templates/packages/vite-template-redux my-app
```

Install:

`./my-app`:
``` shell
npm install
```

Start:

`./my-app`:
``` shell
npm start
```
Then go to <http://localhost:5173/>
