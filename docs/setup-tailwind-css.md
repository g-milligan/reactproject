# Setup Tailwind CSS in Vite

[Documentation](https://tailwindcss.com/docs/guides/vite)

## Install

`./react-crash-2024`
```shell
npm install -D tailwindcss postcss autoprefixer
```

## Init config file

Config files: `postcss.config.js`, `tailwind.config.js`

`./react-crash-2024`
```
npx tailwindcss init -p
```
